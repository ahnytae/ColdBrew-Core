## ColdBrew

ColdBrew 프로젝트는 webRTC를 쉽게 사용하게끔 만든 API 입니다.

React.js / Vue.js / Svelt 등 install 해서 사용 하면 됩니다.

현재는 최소한의 기능이지만 계속해서 기능 추가/ 업데이트 할 계획 입니다.

(화이트보드, mash 방식 등)

## Setup

---

with npm: `npm install coldbrew-core`

with yarn: `yarn add coldbrew-core`
